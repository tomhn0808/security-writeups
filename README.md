# security-writeups
Multiple small projects create to experiment multiple security aspects on my free time.

These projects are intended for educational purposes and may not provide an exhaustive description of the tools used. As they are source of design for students to replicate during their learning process, certain details have been intentionally omitted to encourage independent research.

The projects are:
- DNS Tunneling (using DNS2Cat utility in cleartext mode)
- Fileless malware on Windows (reversel shell (PowerCat) + keylogger (Nishang) + Forensics analysis of volatile memory)
- Captcha Solver (Retrieve a webpage, parse It, use OCR (Optical Character Recognition), send the result to the website and print the response)
- PAM (Pluggable Authentication Modules for Linux )
